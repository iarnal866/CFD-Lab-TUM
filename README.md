## CFD LAB - TUM Advanced Practical Course
C++ scripts developed from fluidchen-skeleton for the CFD Lab Advanced Practical Course at TUM (10 ECTS)

**Group members:** Daniel Cubillo, Thomas Lutz, Patrick Metscher and Icíar Arnal

**Overview**:
CFD solver for 2D incompressible Navier-Stokes equations using SOR pressure solver and adaptive time stepping. 
Implements arbitrary geometries and heat transport across multiple example cases. 

**Status**:
- Completed: SOR solver, considering arbitrary geometries and heat transport .
- In development: MPI parallelization. 
- Final project: TBD.
